# Header 2 broches

## Caractéristiques
| Propriété | Valeur |
|-----------|--------|
| **Désignateur** | H1 |
| **Type** | Connecteur mâle |
| **Pas** | 2.54 mm (0.1") |
| **Boîtier** | HDR-TH_2P-P2.54-V-M-4 |
| **Nombre de broches** | 2 |

---

## Description
Connecteur mâle standardisé avec un pas de 2.54 mm, compatible avec les connecteurs Dupont et les jumpers.

---

## Fonction
- Interface pour **jumpers** ou **câbles Dupont**
- Permet des connexions temporaires ou configurables
- Facilite les tests et le prototypage

---

## Reconnaissance
- Rangée de **2 broches métalliques** dorées ou étamées
- Espacées de 2.54 mm
- Base en plastique noir (généralement)

```
    ┌─┐ ┌─┐
    │ │ │ │  ← Broches métalliques
    └─┘ └─┘
    ▓▓▓▓▓▓▓  ← Base plastique
```

---

## Conseil de montage
Insérez le header verticalement à travers le PCB. Les broches dépasseront du côté soudure. Assurez-vous qu'il soit perpendiculaire au PCB avant de souder.

---

## Utilisation typique
- Configuration de circuits (sélection d'options)
- Points de test
- Connexion de programmateurs
- Alimentation externe
- Connexion de modules
- Jumpers de configuration
